# Privacy Policy

## Overview

This project is a sample API and Copilot plugin integration. It may process prompts, model responses, and metadata for demonstration purposes.

## Data Handling

- Prompt and response data can be stored in configured conversation storage.
- Optional integrations (Azure AI Foundry, Microsoft Graph, Blob Storage, Key Vault, App Insights) process data according to their platform terms.
- Secrets should be stored in secure providers (for example, Key Vault or user-secrets), not source control.

## Responsibility

You are responsible for configuring this project to meet your organization's privacy, retention, and compliance requirements before production use.
